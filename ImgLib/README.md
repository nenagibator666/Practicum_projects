Данная директория являются библеотекой по чтения/записи файлов изображений (`PPM`, `BMP`, `JPEG`).

Файлы `img_lib.h` и `img_lib.cpp` содержат код описания класса `Image`, который используется для дальнейшей конвертации изображений. \
Файлы `bmp_image.h` и `bmp_image.cpp` содержит описание работы функций чтения и записи изображений формата `BMP`. \
Файлы `ppm_image.h` и `ppm_image.cpp` содержит описание работы функций чтения и записи изображений формата `PPM`. \
Файлы `jpeg_image.h` и `jpeg_image.cpp` содержит описание работы функций чтения и записи изображений формата `JPEG`. Для корректной обработки данного формата требуется утсановка библеотеки `jpeglib`.
