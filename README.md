# Структура репозитория #

## Конвертер изображений ##
[Ссылка на проект](https://github.com/nenagibator666/Practicum_projects/tree/master/cpp_image_converter) или перейдите в папку с названием ``cpp_image_converter``\
В данном репозитории находится проект простого конвертора изображений, который способен конвертировать изображения из одного формата в другой. Проект поддерживает такие типы изображений: PPM, BMP, JPEG.
