# Конвертер изображений #
Исходный код конвертера изображений, который поддерживает следующие форматы: `PPM`,` BMP`, `JPG/JPEG`. \
В файлах директории `ImgLib` находится исходный код библиотеки, которая поддерживает запись и загрузку изображений определённого формата (`ppm_image`, `bmp_image`, `jpg_image`).
В файлах директории `ImgConverter` находится код основного файла (`main.cpp`). В данном файле реализован интерфейс для работы с тремя поддерживаемыми форматами изображений. \

Описание проекта: 
* Обработка изображений
* Конвертация форматов изображений
* Чтение/запись изображений

## Описание процесса сборки проекта ##
Для корректной работы данного конвертора требуется установка библиотеки `libjpeg`. \
Для самостоятельной сборки проекта следует использовать следующую последовательность:
1. Скачать и распакавать проект
2. Создать отдельную директорию, в которой будет храниться собранный проект
3. Перейти в созданную директория и выполнить следующую команду:
   ```powershell
   cmake -G "<предпочитаемый генератор>" <директория ImgConverter> -DCMAKE_BUILD_TYPE=Release -DLIBJPEG_DIR=<директория библиотеки libjpeg>
   ```
4. Далее выполнить такую команду:
   ```powershell
   cmake --build .
   ```

После будет создан `imgconv.exe`, который и будет осуществлять конвертацию изображений. \
Пример:
```powershell
.\imgconv.exe <директория исходного изображения> <конечная директория>
```
